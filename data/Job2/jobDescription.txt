JOB TITLE: Angular Developer

COMPANY NAME: WebTech Innovations Inc.

LOCATION: Remote (USA & Canada)  
EMPLOYMENT TYPE: Full-time  
DEPARTMENT: Frontend Engineering  
REPORTS TO: Head of Frontend Development

ABOUT THE COMPANY  
WebTech Innovations Inc. is a leading software development company specializing in modern web applications and digital solutions for enterprise clients across North America. We focus on creating responsive, scalable, and user-friendly web applications that drive business value and enhance user experiences.

POSITION SUMMARY  
WebTech Innovations Inc. is seeking an Angular Developer to join our Frontend Engineering team. This role involves building and maintaining modern web applications using Angular framework, collaborating with UX designers and backend teams, and ensuring high-quality user experiences. The ideal candidate will work closely with cross-functional teams to deliver robust, performant, and visually appealing web applications.

The successful candidate has strong expertise in Angular, TypeScript, and modern frontend development practices, with the ability to write clean, maintainable code and create responsive user interfaces.

KEY RESPONSIBILITIES  
• Develop and maintain web applications using Angular framework (version 12+)  
• Write clean, efficient, and reusable TypeScript code following best practices  
• Build responsive and mobile-friendly user interfaces using HTML5, CSS3, and Angular Material  
• Integrate RESTful APIs and handle asynchronous data operations using RxJS  
• Implement state management solutions using NgRx or similar libraries  
• Collaborate with UX/UI designers to translate designs into functional interfaces  
• Optimize application performance and ensure cross-browser compatibility  
• Write unit tests using Jasmine and Karma, and end-to-end tests using Protractor or Cypress  
• Participate in code reviews and maintain code quality standards  
• Stay current with Angular updates, frontend trends, and best practices  

REQUIRED QUALIFICATIONS  
• Bachelor's degree in Computer Science, Software Engineering, or related field  
• 3–5 years of experience in frontend development with strong focus on Angular  
• Expert knowledge of Angular framework (Angular 12+), TypeScript, and JavaScript  
• Strong proficiency in HTML5, CSS3, and responsive design principles  
• Experience with RxJS for reactive programming and state management  
• Understanding of RESTful API integration and HTTP services  
• Familiarity with version control systems, particularly Git  
• Strong problem-solving and debugging skills  
• Excellent communication and teamwork abilities  

PREFERRED QUALIFICATIONS  
• Experience with NgRx for state management in Angular applications  
• Knowledge of Angular Material or other component libraries  
• Familiarity with build tools like Webpack, npm, and Angular CLI  
• Experience with testing frameworks (Jasmine, Karma, Cypress)  
• Understanding of CI/CD pipelines and DevOps practices  
• Prior experience with agile development methodologies
